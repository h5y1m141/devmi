- 開催日時:2013年3月2日(土)　午後1時30分から午後5時30分

− 主催:パソナテック

## このドキュメントについて

各セッションで取り上げた新しい潮流を踏まえ、2013年の開発現場で求められるエンジニア像について最後のセッションでディスカッションするテーマについてこのドキュメントは扱ってます。

### 業務システムにもオープンソース活用に抵抗なくなったのはいつ頃？（大石さん、安達さん中心にお話をして貰いたい)


私が今の仕事を始めた2005年頃というのは、開発者の求人情報にはOarcleなどの商用製品がキーワードとして多数出てくる一方、オープンソースソフトウェア(以下OSS)を見かけた記憶がほとんどありませんが、最近はこの辺りかなり変化していると感じています。

具体的にいつ頃から、OSSの活用に抵抗なくなったと感じてるのか、

- 総合商社でISP事業などをされて、その後現在の会社設立と業界経験長い大石さん
- 前職TISで社内ベンチャー立ちあげがきっかけで現在の会社でOSSをフル活用している安達さん

という状況を踏まえてお話いただきたい
